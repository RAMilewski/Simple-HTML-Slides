How to Hack The Web


What do we mean when we say "HACK"?



What makes the web unique?


PARTICIPATION:

Anyone can participate or innovate on the web without asking permission from others.

You can see the content and understand it, remix it and make something new.

The web is made up of components that are like Lego building blocks...

...that let you make an infinite number of things.

The world of the web encourages remixing to make new things because that is how it is built.

Why should you
know how to code?

Could you imagine being able to read but not write?


Knowing how to read and write code allows you to be an active participant on this world wide web.

Make sure you have one of these web browsers open:

Go to: www.hackasaurus.org/goggles

Make sure you have your Bookmarks Toolbar displayed:

 View > Toolbars > Bookmarks Toolbar

 View > Show Bookmarks Bar

 View > Always Show Bookmarks Bar

 View > Toolbars > Bookmarks Bar

After you install the goggles, visit a webpage that you would like to hack and click on the "Web X-Ray Goggles" tab in your Bookmarks toolbar

Now, as you move your cursor around the screen, you should be able to see the HTML element information.

Click the "R" key to open up the Remix window.


Look in the code for the part you want to change, and make that edit. The "pretty" view makes it easier for you to see where you need to make changes. Click "Ok" when you're finished.

Now you should see your hacked website.
Press "ESC" to deactivate the goggles or "T" to share your webpage.
