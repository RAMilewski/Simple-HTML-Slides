How to turn a PDF slide deck into a live HTML presentation with synchronized slide changes over the web. (Macintosh)

  • Save the page source for this file as MySlides.html.
  • Open your PDF Slide Deck in Preview.app.
  • Choose File > Print & click on the PDF button.
  • Choose "Save PDF to iPhoto."
  • Open iPhoto and select the album holding your slides.
  • Choose File > Export.
  • Set the following:
    • Kind: JPEG
    • JPEG Quality: Medium
    • Size: Medium (or Large for better quality)
    • File Name: Sequential
    • Prefix for sequential: slide-
  • Click the "Export" button and save to a new directory.
  • Upload the directory of images to a web server.†
  • Edit MySlides.html to make two changes near the end, just after <!-- PDF Slides -->:
    • Set data-source to the path to your images.
    • Set data-count to the number of slides you have.
  • Upload MySlides.html to the web server.†

† If you don't need to synchronize slide changes across multiple machines, no web server is necessary. Simply set the data-source attribute to reference a local directory of images and open MySlides.html in your browser.
